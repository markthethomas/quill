Installation: 

```javascript
λ npm i
```

Run:
```javascript
λ npm i -g babel-cli
λ babel-node ./src/server/server.js
```

Test: 

*continuous*
```javascript
λ npm run test:W
```

*one-off w/ coverage & linting*
```javascript
λ npm test
```

Lint:

```javascript
λ npm run lint
```
